## 5장. 반복하기 — for / while로 일을 자동화하자

---

### 1. 같은 일을 여러 번 하고 싶다

지금까지의 코드는
👉 **한 번 실행하고 끝**이었다.

하지만 현실의 프로그램은
👉 **같은 일을 수십 번, 수백 번 반복**한다.

이때 사용하는 것이 **반복문(loop)** 이다.

---

### 2. for 문 기본 구조

```c
for (초기값; 조건; 증감)
{
    반복 실행할 코드;
}
```

의미를 말로 풀면:

1. 초기값 설정
2. 조건 확인
3. 참이면 코드 실행
4. 증감
5. 다시 조건 확인 → 반복

---

### 3. 가장 기본적인 for 예제

```c
#include <stdio.h>

int main(void)
{
    for (int i = 1; i <= 5; i++)
    {
        printf("%d\n", i);
    }

    return 0;
}
```

출력:

```
1
2
3
4
5
```

---

### 4. 반복 변수 i 이해하기

* `i`는 **카운터**
* 반복 횟수를 세는 용도
* 이름은 꼭 `i`일 필요는 없다

```c
for (int count = 0; count < 3; count++)
{
    printf("Hello\n");
}
```

---

### 5. 10번 인사하기

```c
#include <stdio.h>

int main(void)
{
    for (int i = 0; i < 10; i++)
    {
        printf("Hello, C!\n");
    }

    return 0;
}
```

👉 “코드 1줄로 10번 실행”

---

### 6. while 문 기본 구조

```c
while (조건)
{
    실행할 코드;
}
```

* 조건이 **참인 동안 계속 반복**
* 종료 조건이 없으면 **무한 루프**

---

### 7. while 예제

```c
#include <stdio.h>

int main(void)
{
    int i = 1;

    while (i <= 5)
    {
        printf("%d\n", i);
        i++;
    }

    return 0;
}
```

출력:

```
1
2
3
4
5
```

---

### 8. for vs while 차이

| 구분    | for     | while     |
| ----- | ------- | --------- |
| 반복 횟수 | 정해져 있음  | 모를 때 사용   |
| 구조    | 간결      | 유연        |
| 사용 예  | 1~10 출력 | 입력 받을 때까지 |

---

### 9. 무한 루프 주의

```c
while (1)
{
    printf("끝없이 실행됨\n");
}
```

👉 종료 조건이 없으면 프로그램이 멈추지 않는다.

---

### 10. break — 반복 중단

```c
for (int i = 1; i <= 10; i++)
{
    if (i == 5)
        break;

    printf("%d\n", i);
}
```

출력:

```
1
2
3
4
```

---

### 11. continue — 이번만 건너뛰기

```c
for (int i = 1; i <= 5; i++)
{
    if (i == 3)
        continue;

    printf("%d\n", i);
}
```

출력:

```
1
2
4
5
```

---

### 12. 반복문 + if 조합

```c
#include <stdio.h>

int main(void)
{
    for (int i = 1; i <= 10; i++)
    {
        if (i % 2 == 0)
        {
            printf("%d는 짝수\n", i);
        }
    }

    return 0;
}
```
