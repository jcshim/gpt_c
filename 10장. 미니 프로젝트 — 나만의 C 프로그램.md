## 🧩 예제 프로젝트 ① 숫자 맞추기 게임

이제 하나를 실제로 만들어 보자.

### 🎯 게임 규칙

* 컴퓨터는 1부터 10 사이의 숫자를 하나 정한다
* 사용자는 숫자를 입력한다
* 맞으면 종료, 틀리면 다시 입력

---

## ▶ 코드 작성하기

아래 코드를
**그대로 복사해서 실행**해 보자.

```c
#include <stdio.h>

int main(void)
{
    int answer = 7;
    int guess;

    while (1)
    {
        printf("숫자를 맞혀 보세요 (1~10): ");
        scanf("%d", &guess);

        if (guess == answer)
        {
            printf("정답입니다!\n");
            break;
        }
        else
        {
            printf("틀렸습니다. 다시 시도하세요.\n");
        }
    }

    return 0;
}
```

실행해 보면서
직접 숫자를 바꿔 보자.
