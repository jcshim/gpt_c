## 4장. 판단하기 — if 문으로 생각하는 프로그램 만들기

---

### 1. 프로그램이 “판단”을 하게 하자

지금까지 프로그램은
👉 **그냥 실행만 했다.**

이제부터는
👉 **조건에 따라 다르게 행동**하게 만든다.

이때 사용하는 것이 `if` 문이다.

---

### 2. if 문의 기본 형태

```c
if (조건)
{
    실행할 코드;
}
```

* 조건이 **참(true)** 이면 실행
* 조건이 **거짓(false)** 이면 실행하지 않음

---

### 3. 가장 간단한 if 예제

```c
#include <stdio.h>

int main(void)
{
    int a = 10;

    if (a > 5)
    {
        printf("5보다 큽니다.\n");
    }

    return 0;
}
```

* `a > 5` → 참
* 따라서 문장이 출력된다

---

### 4. 입력값으로 판단하기

```c
#include <stdio.h>

int main(void)
{
    int a;

    scanf("%d", &a);

    if (a > 0)
    {
        printf("양수입니다.\n");
    }

    return 0;
}
```

입력:

```
3
```

출력:

```
양수입니다.
```

---

### 5. if ~ else 구조

```c
if (조건)
{
    참일 때 실행
}
else
{
    거짓일 때 실행
}
```

---

### 6. if ~ else 예제

```c
#include <stdio.h>

int main(void)
{
    int a;

    scanf("%d", &a);

    if (a >= 60)
    {
        printf("합격\n");
    }
    else
    {
        printf("불합격\n");
    }

    return 0;
}
```

---

### 7. 비교 연산자 정리

| 연산자  | 의미  |
| ---- | --- |
| `>`  | 크다  |
| `<`  | 작다  |
| `>=` | 이상  |
| `<=` | 이하  |
| `==` | 같다  |
| `!=` | 다르다 |

⚠️ `=` 와 `==` 는 완전히 다르다.

---

### 8. 가장 많이 하는 실수

❌ 잘못된 코드

```c
if (a = 10)
```

* 대입 연산
* 항상 참이 됨

⭕ 올바른 코드

```c
if (a == 10)
```

---

### 9. 여러 조건 판단하기 (else if)

```c
#include <stdio.h>

int main(void)
{
    int score;
    scanf("%d", &score);

    if (score >= 90)
        printf("A\n");
    else if (score >= 80)
        printf("B\n");
    else if (score >= 70)
        printf("C\n");
    else
        printf("F\n");

    return 0;
}
```

---

### 10. 중괄호는 습관처럼 쓰자

```c
if (a > 0)
    printf("양수\n");
    printf("끝\n");
```

👉 실제로는 항상 `"끝"`이 출력된다.

안전한 습관:

```c
if (a > 0)
{
    printf("양수\n");
}
```
