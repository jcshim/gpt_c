## 3장. 입력받기 — 키보드로 숫자를 넣어보자

---

### 1. 프로그램과 “대화”하기

지금까지는
👉 **우리가 미리 정한 값만 출력**했다.

이제는
👉 **사용자가 키보드로 값을 입력**하게 해 보자.

---

### 2. scanf — 입력받는 함수

C언어에서 입력은 `scanf` 를 사용한다.

```c
scanf("%d", &a);
```

* `%d` : 정수를 입력받겠다
* `a` : 변수 이름
* `&a` : **변수의 주소**

👉 **입력은 반드시 &를 붙인다**

---

### 3. 가장 기본적인 입력 예제

```c
#include <stdio.h>

int main(void)
{
    int a;

    scanf("%d", &a);
    printf("%d\n", a);

    return 0;
}
```

실행 흐름:

1. 프로그램 실행
2. 숫자 입력 후 Enter
3. 입력한 숫자가 그대로 출력됨

---

### 4. 입력 전에 안내 문구 출력하기

```c
#include <stdio.h>

int main(void)
{
    int a;

    printf("숫자를 입력하세요: ");
    scanf("%d", &a);

    printf("입력한 값은 %d입니다.\n", a);
    return 0;
}
```

출력 예:

```
숫자를 입력하세요: 7
입력한 값은 7입니다.
```

---

### 5. 두 개의 숫자 입력받기

```c
#include <stdio.h>

int main(void)
{
    int a, b;

    scanf("%d %d", &a, &b);
    printf("%d\n", a + b);

    return 0;
}
```

입력:

```
3 5
```

출력:

```
8
```

---

### 6. scanf에서 가장 많이 하는 실수

❌ 잘못된 코드

```c
scanf("%d", a);
```

* `&` 빠짐
* 컴파일 에러 또는 실행 오류

⭕ 올바른 코드

```c
scanf("%d", &a);
```

---

### 7. 왜 &를 붙일까?

* 변수 `a` → 값이 저장된 **메모리 위치**
* `&a` → 그 **주소**

`scanf`는
👉 **“어디에 값을 저장할지”** 알아야 한다.

그래서 주소가 필요하다.

---

### 8. 입력 → 계산 → 출력 한 번에

```c
#include <stdio.h>

int main(void)
{
    int x, y;

    printf("두 수 입력: ");
    scanf("%d %d", &x, &y);

    printf("합: %d\n", x + y);
    return 0;
}
```
