# 3장. 숫자를 다뤄보자 — 변수와 계산

지금까지 우리는 이런 사실을 확인했다.

* C 프로그램은 `main` 함수에서 시작된다
* 코드는 위에서 아래로 순서대로 실행된다

이제 본격적으로 **숫자**를 다뤄보자.
컴퓨터에게 일을 시키려면, 숫자를 기억하고 계산할 수 있어야 한다.

하지만 또다시 외우지 않는다.
**먼저 GPT에게 묻는다.**

---

## 💬 GPT에게 이렇게 물어보자

> **“프로그래밍에서 변수란 무엇인가요?
> 초등학생도 이해할 수 있게 설명해 주세요.”**

GPT는 보통 이렇게 설명할 것이다.

* 변수는 값을 저장하는 상자다
* 상자에는 이름이 있다
* 그 안에 숫자를 넣었다가 다시 꺼내 쓴다

여기서 중요한 것은 딱 하나다.

> **컴퓨터는 숫자를 ‘기억’하지 못한다.
> 대신 변수라는 상자에 담아 둔다.**

---

## 📦 숫자를 담는 상자, 변수

C언어에서 숫자를 담는 가장 기본적인 상자는 두 가지다.

* `int`  : 정수 (소수점 없는 숫자)
* `float` : 실수 (소수점 있는 숫자)

지금은 이 두 가지만 사용한다.

---

## ▶ 첫 번째 변수 사용해 보기

아래 코드를 **그대로 입력하고 실행**해 보자.

```c
#include <stdio.h>

int main(void)
{
    int a = 10;
    int b = 3;

    printf("a = %d\n", a);
    printf("b = %d\n", b);

    return 0;
}
```

실행 결과는 다음과 같다.

```
a = 10
b = 3
```

여기서 확인할 수 있는 사실은 이것이다.

> **변수는 숫자를 저장하고,
> 이름으로 다시 꺼내 쓸 수 있다.**

---

## ➕ 숫자를 계산해 보자

이제 변수를 이용해 계산을 해보자.

```c
#include <stdio.h>

int main(void)
{
    int x = 10;
    int y = 3;

    int sum = x + y;
    int sub = x - y;
    int mul = x * y;
    int div = x / y;

    printf("덧셈: %d\n", sum);
    printf("뺄셈: %d\n", sub);
    printf("곱셈: %d\n", mul);
    printf("나눗셈: %d\n", div);

    return 0;
}
```

실행 결과를 확인해 보자.

```
덧셈: 13
뺄셈: 7
곱셈: 30
나눗셈: 3
```

여기서 한 가지 이상한 점이 있다.

> **10 ÷ 3 = 3 ?**

---

## 🤔 왜 나눗셈 결과가 3일까?

이 질문도 GPT에게 던져 보자.

---

## 💬 GPT에게 이렇게 물어보자

> **“C언어에서 10 / 3을 하면 왜 3이 나오나요?”**

GPT는 이렇게 설명할 것이다.

* `int`는 정수만 저장한다
* 소수점 아래는 버려진다
* 그래서 결과도 정수가 된다

즉,

> **정수끼리 계산하면 결과도 정수다.**

---

## 🔢 소수점 계산을 해보자 (float)

이번에는 `float`를 사용해 보자.

```c
#include <stdio.h>

int main(void)
{
    float a = 10.0f;
    float b = 3.0f;

    float result = a / b;

    printf("결과: %f\n", result);

    return 0;
}
```

실행 결과는 다음과 비슷하게 나온다.

```
결과: 3.333333
```

여기서 알 수 있는 핵심은 이것이다.

> **어떤 상자를 쓰느냐에 따라
> 계산 결과가 달라진다.**

---

## 😵‍💫 초보자가 자주 하는 실수

### ❌ 정수와 실수를 헷갈리는 경우

```c
int a = 10;
int b = 3;
float c = a / b;
```

이 코드는 소수점 결과를 주지 않는다.
**이미 계산이 정수로 끝났기 때문이다.**

문제가 생기면, 그대로 GPT에게 보여 주자.
