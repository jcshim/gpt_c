# 4장. 입력받아 보자 — scanf 이해하기

지금까지 우리는 숫자를 **코드 안에 직접 써서** 계산했다.  
하지만 실제 프로그램은 이렇게 묻는다.

> “사용자가 직접 값을 넣게 할 수는 없을까?”

이 질문 역시 혼자 고민하지 말고,  
**GPT에게 먼저 던져 보자.**

---

## 💬 GPT에게 이렇게 물어보자

> **“C언어에서 키보드로 숫자를 입력받으려면 어떻게 하나요?”**

GPT는 보통 이런 답을 준다.

- `scanf` 함수를 사용한다  
- 키보드 입력을 변수에 저장할 수 있다  
- 입력 형식이 중요하다  

지금은 이 정도만 알면 충분하다.

> **scanf는  
> ‘사용자가 입력한 값을 변수에 넣어주는 도구’다.**

---

## ⌨️ 키보드로 숫자 입력받기

아래 코드를 **그대로 입력하고 실행**해 보자.

```c
#include <stdio.h>

int main(void)
{
    int age;

    printf("나이를 입력하세요: ");
    scanf("%d", &age);

    printf("입력한 나이: %d\n", age);

    return 0;
}
````

실행하면 다음과 같은 흐름이 된다.

1. 화면에 문장이 출력된다
2. 커서가 깜빡인다
3. 숫자를 입력하고 Enter를 누른다

예를 들어 `20`을 입력하면,

```
나이를 입력하세요: 20
입력한 나이: 20
```

이렇게 출력된다.

## 📌 여기서 가장 중요한 한 줄

```c
scanf("%d", &age);
```

이 줄에서 **절대 그냥 넘어가면 안 되는 부분**이 있다.

바로 이것이다.

```c
&age
```

---

## 🤔 왜 변수 앞에 &를 붙일까?

이 질문도 그대로 GPT에게 던져 보자.

---

## 💬 GPT에게 이렇게 물어보자

> **“scanf에서 왜 변수 앞에 &를 붙이나요?”**

GPT는 이렇게 설명해 줄 것이다.

* 컴퓨터는 값이 아니라 **위치**에 써야 한다
* `&`는 변수의 주소를 의미한다
* scanf는 그 주소에 값을 채워 넣는다

지금 단계에서는 이렇게만 기억하자.

> **scanf는
> ‘변수의 주소’를 알아야 값을 넣을 수 있다.**

그래서 `&`가 필요하다.

---

## 🧪 두 개의 숫자 입력받아 계산하기

이번에는 두 개의 숫자를 입력받아
합을 계산해 보자.

```c
#include <stdio.h>

int main(void)
{
    int a, b;

    printf("두 개의 숫자를 입력하세요: ");
    scanf("%d %d", &a, &b);

    printf("합: %d\n", a + b);

    return 0;
}
```

실행 예시는 다음과 같다.

```
두 개의 숫자를 입력하세요: 7 5
합: 12
```

여기서 중요한 사실 하나를 알 수 있다.

> **scanf는 여러 값을 한 번에 입력받을 수도 있다.**

## 😵‍💫 초보자가 가장 많이 하는 실수

### ❌ &를 빼먹는 경우

```c
scanf("%d", age);
```

이 코드는 거의 항상 문제를 일으킨다.

* 실행 중 오류가 나거나
* 이상한 값이 들어가거나
* 프로그램이 멈춘다

이럴 때는 절대 혼자 끙끙대지 말고,
그대로 GPT에게 보여 주자.
