# 9장. 눈으로 보는 C언어 — raylib 맛보기

지금까지 우리가 만든 모든 C 프로그램은
검은 화면에 글자만 출력했다.

하지만 이런 생각이 들기 시작한다.

> “C언어로
> 진짜 프로그램 같은 걸
> 만들 수는 없을까?”

이 질문에 대한 가장 좋은 대답은
**눈으로 보이는 결과**다.

---

## 💬 GPT에게 이렇게 물어보자

> **“C언어로도
> 화면에 그림을 그릴 수 있나요?”**

GPT는 이렇게 답할 것이다.

* 가능하다
* 그래픽 라이브러리를 사용하면 된다
* raylib 같은 도구가 있다

지금 이 장에서는
**raylib을 깊이 배우지 않는다.**

> **‘맛보기’만 한다.**

---

## 🎨 raylib은 무엇인가?

raylib은
C언어로 그래픽을 쉽게 다룰 수 있게 해 주는 도구다.

* 복잡한 설정 없이
* 짧은 코드로
* 바로 결과를 볼 수 있다

이 장의 목표는 단 하나다.

> **“와, C언어로 이런 것도 되네.”**

---

## 🧰 raylib 설치 (NuGet)

Visual Studio에서
raylib을 가장 간단하게 사용하는 방법은
**NuGet**이다.

아래 순서는
이해하지 말고 그대로 따라 하자.

1. 프로젝트 우클릭
2. **NuGet 패키지 관리** 선택
3. `raylib` 검색
4. **설치** 클릭
5. 끝

이 단계에서
“왜 이렇게 되는지”는
지금 알 필요 없다.

---

## ▶ 첫 번째 raylib 프로그램

아래 코드를
**그대로 복사해서 실행**해 보자.

```c
#include "raylib.h"

int main(void)
{
    InitWindow(400, 300, "Hello raylib!");

    while (!WindowShouldClose())
    {
        BeginDrawing();
        ClearBackground(RAYWHITE);
        DrawText("Hello, C Graphics!", 60, 140, 20, DARKBLUE);
        EndDrawing();
    }

    CloseWindow();
    return 0;
}
```

실행하면
새 창이 하나 뜬다.

* 마우스로 창을 움직일 수 있고
* 글자가 화면에 보인다

이 순간이
이 책에서 가장 중요한 순간 중 하나다.

---

## 🎉 무엇을 확인했을까?

아직 이 코드의 모든 줄을
이해하지 않아도 괜찮다.

지금 확인한 것은 이것이다.

* C언어로 창을 열 수 있다
* 화면에 글자를 그릴 수 있다
* 반복문으로 화면을 계속 갱신한다

> **C언어는
> 단순한 계산 언어가 아니다.**
