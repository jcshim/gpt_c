## 7장. 함수(function) — 코드를 묶어서 다시 쓰자

---

### 1. 왜 함수가 필요할까?

같은 코드가 계속 반복된다면
👉 **복사–붙여넣기**를 하게 된다.

함수는
👉 **코드를 한 번만 만들어 여러 번 쓰기** 위한 장치다.

---

### 2. 함수의 기본 형태

```c
반환형 함수이름(매개변수)
{
    실행할 코드;
    return 반환값;
}
```

---

### 3. 가장 단순한 함수 예제

```c
#include <stdio.h>

void hello(void)
{
    printf("Hello C!\n");
}

int main(void)
{
    hello();
    hello();
    return 0;
}
```

* `void` : 반환값 없음
* 함수는 **위에서 정의 → 아래에서 호출**

---

### 4. 함수 선언과 정의

```c
#include <stdio.h>

int add(int a, int b);   // 함수 선언

int main(void)
{
    printf("%d\n", add(3, 5));
    return 0;
}

int add(int a, int b)    // 함수 정의
{
    return a + b;
}
```

👉 선언은 **“이런 함수가 있다”**
👉 정의는 **“실제로 어떻게 동작하는지”**

---

### 5. 매개변수(Parameter)와 인자(Argument)

```c
int add(int a, int b)
```

* `a`, `b` → 매개변수
* `add(3, 5)`의 `3`, `5` → 인자

---

### 6. 반환값(return)

```c
int square(int x)
{
    return x * x;
}
```

* `return`은 **값을 돌려주고 함수 종료**
* 반환형과 return 값의 타입은 같아야 한다

---

### 7. Call by Value (값 전달)

```c
#include <stdio.h>

void change(int x)
{
    x = 100;
}

int main(void)
{
    int a = 10;
    change(a);
    printf("%d\n", a);
    return 0;
}
```

출력 결과: `10`

👉 함수는 **값의 복사본**을 받는다.

---

### 8. 함수는 작은 단위가 좋다

나쁜 예:

```c
void program() { /* 너무 많은 코드 */ }
```

좋은 예:

```c
void input();
void process();
void output();
```

---

### 9. 함수 분리 예제

```c
#include <stdio.h>

int add(int a, int b)
{
    return a + b;
}

void printResult(int result)
{
    printf("결과: %d\n", result);
}

int main(void)
{
    int r = add(4, 6);
    printResult(r);
    return 0;
}
```
