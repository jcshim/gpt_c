## 6장. 조건 분기 — if / else로 판단하게 만들자

---

### 1. 프로그램이 스스로 판단하게 하려면

지금까지 프로그램은
👉 **정해진 순서대로만 실행**했다.

이제부터는
👉 **조건에 따라 다른 행동**을 하게 만든다.

그 역할을 하는 것이 **if 문**이다.

---

### 2. if 문 기본 구조

```c
if (조건)
{
    조건이 참일 때 실행할 코드;
}
```

* 조건이 **참(true)** 이면 실행
* 거짓(false)이면 실행하지 않음

---

### 3. 가장 간단한 if 예제

```c
#include <stdio.h>

int main(void)
{
    int score = 80;

    if (score >= 60)
    {
        printf("합격\n");
    }

    return 0;
}
```

---

### 4. 비교 연산자 정리

| 연산자  | 의미  |
| ---- | --- |
| `>`  | 크다  |
| `<`  | 작다  |
| `>=` | 이상  |
| `<=` | 이하  |
| `==` | 같다  |
| `!=` | 다르다 |

⚠️ `=` 와 `==` 는 완전히 다르다.

---

### 5. if – else 구조

```c
if (조건)
{
    참일 때 실행;
}
else
{
    거짓일 때 실행;
}
```

---

### 6. 합격 / 불합격 예제

```c
#include <stdio.h>

int main(void)
{
    int score = 55;

    if (score >= 60)
        printf("합격\n");
    else
        printf("불합격\n");

    return 0;
}
```

---

### 7. else if — 여러 조건 처리

```c
#include <stdio.h>

int main(void)
{
    int score = 85;

    if (score >= 90)
        printf("A\n");
    else if (score >= 80)
        printf("B\n");
    else if (score >= 70)
        printf("C\n");
    else
        printf("F\n");

    return 0;
}
```

---

### 8. 조건식은 참/거짓이다

C언어에서는

* **0 → 거짓**
* **0이 아닌 값 → 참**

```c
if (1)  // 항상 실행
if (0)  // 절대 실행 안 됨
```

---

### 9. 논리 연산자

| 연산자  | 의미        |   |         |
| ---- | --------- | - | ------- |
| `&&` | 그리고 (AND) |   |         |
| `    |           | ` | 또는 (OR) |
| `!`  | 아니다 (NOT) |   |         |

---

### 10. 논리 연산자 예제

```c
#include <stdio.h>

int main(void)
{
    int age = 20;
    int student = 1;

    if (age >= 19 && student == 1)
    {
        printf("대학생\n");
    }

    return 0;
}
```

---

### 11. 중괄호 생략 주의

```c
if (x > 0)
    printf("양수\n");
    printf("항상 실행됨\n");
```

👉 두 번째 printf는 if와 **무관**

초보자는 **항상 중괄호 사용** 권장.

---

### 12. 조건문 + 반복문 조합

```c
#include <stdio.h>

int main(void)
{
    for (int i = 1; i <= 10; i++)
    {
        if (i % 3 == 0)
        {
            printf("%d\n", i);
        }
    }

    return 0;
}
```
