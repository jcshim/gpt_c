## 8장. 배열과 문자열 — 여러 값을 한 번에 다루기

---

### 1. 배열이 왜 필요할까?

변수가 하나일 때는 쉽다.

```c
int a = 10;
```

그런데 값이 10개, 100개라면?

```c
int a1, a2, a3, a4, a5, a6, a7, a8, a9, a10;
```

👉 **배열(Array)** 은
👉 **같은 종류의 데이터를 한 줄로 묶는 방법**이다.

---

### 2. 배열의 기본 선언

```c
int a[5];
```

* `a` : 배열 이름
* `5` : 칸의 개수
* 인덱스는 **0부터 시작**

---

### 3. 배열에 값 넣기

```c
int a[5] = {1, 2, 3, 4, 5};
```

각 칸의 위치:

```
a[0] a[1] a[2] a[3] a[4]
```

---

### 4. 배열 값 출력하기

```c
#include <stdio.h>

int main(void)
{
    int a[5] = {10, 20, 30, 40, 50};

    for (int i = 0; i < 5; i++)
    {
        printf("%d ", a[i]);
    }
    return 0;
}
```

👉 배열은 **for문과 궁합이 최고**

---

### 5. 배열의 크기 주의

```c
int a[3];
a[3] = 100;   // ❌ 오류 (범위 초과)
```

* 마지막 인덱스는 `크기 - 1`
* 범위 초과는 **치명적인 버그**

---

### 6. 문자열은 문자 배열이다

```c
char s[6] = "Hello";
```

실제 저장 모습:

```
'H' 'e' 'l' 'l' 'o' '\0'
```

👉 문자열 끝에는 **반드시 `'\0'` (널 문자)**

---

### 7. 문자열 출력

```c
#include <stdio.h>

int main(void)
{
    char s[] = "C language";
    printf("%s\n", s);
    return 0;
}
```

* `%s` : 문자열 출력
* 배열 크기를 생략하면 자동 계산

---

### 8. 문자열 한 글자씩 접근

```c
char s[] = "apple";

for (int i = 0; s[i] != '\0'; i++)
{
    printf("%c ", s[i]);
}
```

---

### 9. 문자열 입력 (주의!)

```c
char name[20];
scanf("%s", name);
```

* 공백 전까지 입력
* 크기 초과 입력은 위험

👉 지금 단계에서는 **짧은 문자열만 사용**

---

### 10. 배열을 함수에 전달하기

```c
#include <stdio.h>

void printArray(int a[], int size)
{
    for (int i = 0; i < size; i++)
        printf("%d ", a[i]);
}

int main(void)
{
    int x[3] = {1, 2, 3};
    printArray(x, 3);
    return 0;
}
```

👉 배열 이름은 **첫 번째 요소의 주소**

---

### 11. 문자열 배열 (문자열 여러 개)

```c
char *fruits[] = { "apple", "banana", "kiwi" };
```

* 문자열들의 배열
* 실제 프로젝트에서 매우 자주 사용

---

### 12. 배열 + 함수 = 실전 핵심

```c
int sum(int a[], int n)
{
    int s = 0;
    for (int i = 0; i < n; i++)
        s += a[i];
    return s;
}
```
