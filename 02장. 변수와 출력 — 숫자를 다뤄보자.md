## 2장. 변수와 출력 — 숫자를 다뤄보자

---

### 1. 변수는 “값에 붙이는 이름표”다

C언어에서 **변수(variable)** 는
값을 저장하는 **상자**이자, 그 상자에 붙인 **이름표**다.

```c
int a = 10;
```

* `int` : 정수(integer)
* `a` : 변수 이름
* `10` : 저장할 값

> 의미:
> “정수 하나를 저장할 상자를 만들고, 이름을 a로 붙인 뒤 10을 넣는다.”

---

### 2. 변수를 출력해 보자

```c
#include <stdio.h>

int main(void)
{
    int a = 10;
    printf("%d\n", a);
    return 0;
}
```

실행 결과:

```
10
```

`%d` 는 **정수 출력용 표시자(format)** 이다.

---

### 3. 변수 값은 바꿀 수 있다

```c
int a = 10;
a = 20;
```

* 처음에는 10
* 그 다음에는 20

항상 **마지막으로 대입한 값이 변수의 값**이다.

---

### 4. 여러 개의 변수를 써 보자

```c
#include <stdio.h>

int main(void)
{
    int x = 3;
    int y = 7;

    printf("%d\n", x);
    printf("%d\n", y);

    return 0;
}
```

출력:

```
3
7
```

---

### 5. 변수로 계산하기

```c
#include <stdio.h>

int main(void)
{
    int a = 10;
    int b = 20;
    int sum = a + b;

    printf("%d\n", sum);
    return 0;
}
```

출력:

```
30
```

C언어는 **계산기처럼 생각하면 된다.**

---

### 6. 출력 형식 한 번에 여러 개

```c
printf("%d + %d = %d\n", a, b, sum);
```

출력:

```
10 + 20 = 30
```

* `%d` 개수와
* 변수 개수는 **반드시 같아야 한다**

---

### 7. GPT에게 이렇게 물어보면 좋다

> 💬 “int 말고 다른 자료형에는 뭐가 있나요?”
> 💬 “왜 %d를 쓰나요?”

지금은 전부 외우지 않아도 된다.
**궁금할 때마다 질문하는 습관**이 중요하다.

---

### 8. 초보자가 가장 많이 틀리는 부분

❌ 잘못된 코드

```c
int a;
printf("%d\n", a);
```

* 값이 초기화되지 않았다
* 출력 결과는 **예측 불가**

⭕ 올바른 코드

```c
int a = 0;
printf("%d\n", a);
```
